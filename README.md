# Face Recognition using OpenCV

A real-time face detection and tracking project built using Python and OpenCV.

## Features
- Real-time webcam face detection
- Face movement tracking
- Rectangle face detection
- Haar Cascade classifier integration

## Technologies Used
- Python
- OpenCV
- Haarcascade XML

## Project Structure

```bash
FACERECOGNITION/
│
├── face_recognition.py
├── haarcascade_frontalface_default.xml
├── README.md
```

## Installation

Install OpenCV:

```bash
pip install opencv-python
```

## Run the Project

```bash
python face_recognition.py
```

Press `ESC` to close the webcam window.
